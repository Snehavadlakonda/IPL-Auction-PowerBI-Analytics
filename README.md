
# IPL Auction Analytics Dashboard (Power BI)

## Project Overview
This project analyzes IPL auction data from 2013–2022 using Microsoft Power BI to uncover insights about franchise spending, player valuation, and auction trends.

## Dataset
- 543 players
- 10 IPL seasons
- 13 franchises

## Key Insights
- Total auction spending: ₹2.04K Cr
- Highest bid: ₹16.25 Cr (Christopher Morris)
- All-Rounders received the highest budget share (38.68%)
- Punjab Kings recorded the highest franchise spending.

## Dashboards Created
1. Main Auction Overview
2. Player Analytics Dashboard
3. Franchise Spending Analysis
4. Price Bracket Distribution

## Tools Used
- Microsoft Power BI
- Data Visualization
- DAX Measures
- Data Analysis
